# EboUPrime Konfiguration
EboUPrime ist eine Software, die für die Verwaltung und Konfiguration von EboUPrime-Geräten entwickelt wurde. Diese Dokumentation bietet eine umfassende Anleitung zur Installation, Konfiguration und Nutzung der Software.

## Installation
```bash
pip install eup_config
```
Alternative Installation

```bash
python -m pip install git+https://github.com/EboUPrime/eup_config.git
```

## Aktualisierung
```bash
pip install --upgrade eup_config
```
Alternative Aktualisierung

```bash
python -m pip install --upgrade git+https://github.com/EboUPrime/eup_config.git
```
